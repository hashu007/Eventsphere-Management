# EventSphere Management System

   A comprehensive MERN stack application for managing expos, trade shows, and exhibitions.

   ## Features

   - User authentication with JWT
   - Role-based access (Admin, Exhibitor, Attendee)
   - Expo creation and management
   - Booth allocation system
   - Session registration
   - Profile management

   ## Tech Stack

   - MongoDB
   - Express.js
   - React
   - Node.js

   ## Installation

   ### Backend Setup
```bash
   cd backend
   npm install
   npm run dev
```

   ### Frontend Setup
```bash
   cd frontend
   npm install
   npm start
```
